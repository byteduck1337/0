# /0byte/ — Zero‑Server Private Messenger

**/0byte/** — это полностью клиентский, peer‑to‑peer (P2P) мессенджер с end‑to‑end шифрованием (E2EE). Для обмена сообщениями и файлами не требуется центральный сервер: соединение устанавливается напрямую между браузерами через WebRTC. Передача ключей шифрования и информации о подключении (SDP) происходит вручную, что даёт пользователю полный контроль над приватностью.

## 🚀 Возможности

- 🔒 **E2E шифрование** — все сообщения и изображения шифруются алгоритмом **AES-256-GCM**.
- 🌐 **Полностью без сервера** — WebRTC + ручной обмен SDP, никаких посредников.
- 🎨 **Адаптивный дизайн** — светлая и тёмная тема, поддержка мобильных устройств.
- 🖼️ **Отправка изображений** — автоматическое сжатие до 800px и шифрование.
- 📌 **Закреплённые сообщения** — важные сообщения можно сохранить в панели чата.
- 🔐 **Визуальная маскировка SDP** — технические данные превращаются в мнемонические слова для удобства ручной передачи.
- 💾 **Локальное хранение истории** — вся переписка хранится только на вашем устройстве (`localStorage`).
